1. Crop Prediction Request- 

Postman Setup:

Method: POST
URL: http://127.0.0.1:5000/crop-predict
Headers:
Key: Content-Type, Value: application/json
Body:
{
  "nitrogen": 50,
  "phosphorous": 50,
  "pottasium": 50,
  "ph": 7,
  "rainfall": 200,
  "city": "Bhopal"
}

2. Disease Prediction Request
Postman Setup:

Method: POST
URL: http://127.0.0.1:5000/disease-predict
Body:
Select form-data
Key: file, Value: select File and upload some.jpg from the images directory on your machine.
